# EGESHER

EGESHER — это веб-приложение для подготовки к ЕГЭ, где пользователи могут просматривать задания по школьным предметам, публиковать свои задачи и обсуждать решения в комментариях.

## Основные возможности

- регистрация и авторизация пользователей;
- просмотр заданий по предметам (русский язык, математика, физика, информатика);
- добавление новых заданий с текстовым описанием и изображениями;
- хранение ответов и кодовых решений;
- комментирование заданий.

## Технологии

- Python
- Flask
- Flask-Login
- Flask-RESTful
- SQLAlchemy
- WTForms

## Запуск проекта

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Запустите приложение:
   ```bash
   python server.py
   ```

Также можно использовать скрипт запуска:

```bash
bash start.sh
```
